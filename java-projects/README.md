## 🌳 Decision Tree Classifier

> [`decision-tree-classifier/`](decision-tree-classifier/)

A binary decision tree trained by greedily minimising weighted Shannon entropy at each split. Supports any number of attributes, configurable minimum node size (depth control), Java object serialization for saving/loading models, and recursive structural equality comparison.

**Highlights:**
- No ML libraries — entropy, splits, and tree traversal implemented from scratch
- Achieves **200/200 correct** on training data across all included datasets
- Datasets range from 2-attribute linearly-separable to 5-attribute overlapping

**Key concepts:** CART-style greedy split, information entropy, recursive tree construction, `ObjectOutputStream` serialization, train/test split.

## 🃏 Solitaire Cipher

> [`solitaire-cipher/`](solitaire-cipher/)

Bruce Schneier's card-based stream cipher implemented using a **circular doubly-linked list**. The deck of 54 cards (52 playing + 2 jokers) acts as the shared secret key. Encode and decode use a keystream generated by triple-cut, count-cut, and look-up operations on the deck.

**Highlights:**
- All 8 deck operations run directly on `prev`/`next` pointers — no array copies
- Correct **Fisher-Yates shuffle** (O(n), in-place) with full pointer rebuild
- **24/24 automated tests** covering pointer integrity, edge cases, and deep copy correctness

**Key concepts:** Circular doubly-linked list, deep copy constructor, in-place shuffle, stream cipher, inner class polymorphism (`Card` → `PlayingCard` / `Joker`).

## ♟️ Chess Sudoku Solver

> [`chess-sudoku-solver/`](chess-sudoku-solver/)

A recursive backtracking Sudoku solver extended with optional chess-piece movement constraints. Supports **Knight**, **King**, and **Queen** rules individually or combined, and grid sizes from 9×9 up to 25×25. Can return a single solution or enumerate every valid solution.

**Highlights:**
- Solves 22 included puzzle files; 3×3 puzzles finish in < 3 ms
- Chess constraints reduce the solution space significantly on knight/king/queen puzzles
- Optional Swing-based **GUI visualiser** shows the board state live

**Key concepts:** Recursive backtracking, constraint propagation, chess movement geometry, `HashSet` solution enumeration, Swing visualizer.

## 🛠 Quick Start

### Prerequisites

- Java 8 or higher (`java -version`)

### Compile & run any project

```bash
# Decision Tree Classifier
cd java-projects/decision-tree-classifier
javac -d out src/*.java
java -cp out Tester

# Solitaire Cipher
cd java-projects/solitaire-cipher
javac -d out src/assignment2/*.java
java -cp out assignment2.A2_Minitester_Updated3

# Chess Sudoku Solver
cd java-projects/chess-sudoku-solver/src
javac -d . *.java
java finalproject.Tester
```

See each project's own `README.md` for full usage instructions, API reference, and dataset/puzzle details.
